## Quick Installation

git clone https://github.com/addupy/AI-Powered-WA-Bot.git
cd AI-Powered-WA-Bot
npm install
cp .env.example .env
nano .env
npm start

Then scan QR Code with WhatsApp: Settings > Linked Devices > Link a Device
