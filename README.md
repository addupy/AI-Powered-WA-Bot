## 📦 **Quick Installation**

### Prerequisites
- Node.js 18.x or higher
- npm or yarn
- WhatsApp account (secondary recommended)
- Git

### Setup

```bash
# 1. Clone the repository
git clone https://github.com/addupy/AI-Powered-WA-Bot.git
cd AI-Powered-WA-Bot

# 2. Install dependencies
npm install

# 3. Configure environment
cp .env.example .env
nano .env

# 4. Fill in your details
# BOT_NAME=YourBotName
# BOT_PREFIX=!
# OWNER_NUMBER=your_whatsapp_number
# OPENAI_API_KEY=your_openai_key
# GEMINI_API_KEY=your_gemini_key
# DEEPSEEK_API_KEY=your_deepseek_key
# ENCRYPTION_KEY=your_32_char_key

# 5. Start the bot
npm start

# 6. Scan QR Code with WhatsApp
# Settings → Linked Devices → Link a Device
```

🔐 Security Features Deep Dive

Authentication Flow

```
User Message → Owner Check → Spam Check → Rate Limit → Execute
                     ↓              ↓            ↓
                  Blocked       Warning      Cooldown
```

Encryption Standards

· Algorithm: AES-256-GCM
· Key Derivation: PBKDF2 with 10000 iterations
· Password Hashing: SHA-512 with 32-byte salt
· Session Tokens: SHA-256 with JWT secret

Anti-Abuse System

Protection Threshold
Rate Limit 30 req/min per command
Spam Detection 5 msgs in 10 sec = Warning
Auto Block 3 warnings = Permanent block
Login Attempts 5 fails = 30 min lockout
